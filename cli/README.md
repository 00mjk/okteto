# cli

Command line tool to manage sandboxes on demand

## Installation

```
git clone git@github.com:okteto/app.git
```

We use `go mod`, so you can check out the repo anywhere. 

To build it for macos, run:
```
make build
```

To build for all supports OSs, run:
```
make -j 3 build-all
```

To run tests:
```
make test
```
